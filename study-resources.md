# Study resources

  GitHub Docs – Markdown Guide
  Official and beginner-friendly. Covers basics like headings, lists, links, images, code blocks, tables, and task lists. Best place to learn how Markdown works *specifically on GitHub*.

  The Markdown Guide (markdownguide.org)
  Clear explanations with examples. Great for understanding core Markdown syntax before using it on GitHub.

  GitHub Markdown Cheat Sheets
  One-page references showing common syntax (bold, italics, code, tables, checkboxes). Useful for quick lookups while writing README files.

  Practice on GitHub README files
  Editing a README.md in a repo is the fastest way to learn. GitHub’s preview tab lets students see results instantly.

  Online Markdown Editors
  Tools like Dillinger or StackEdit help students practice Markdown before pushing to GitHub.
